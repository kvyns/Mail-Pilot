# Mail Pilot - Modern CRM Platform

Complete enterprise-grade CRM built with React + Vite + Tailwind CSS

## Features
- Landing Page with modern SaaS design
- Auth (Login/Register with demo account)
- Dashboard with stats and analytics
- Users Management (CSV/Excel import)
- Email Campaigns (create, send, track)
- Drag-and-Drop Email Builder
- Template Management
- Credit-Based System
- Settings Page
